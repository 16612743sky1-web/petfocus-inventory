# 작업 로그 — 상세페이지 디자인 시스템 (2026-07-24)

> 세션이 초기화돼도 맥락이 남도록, 지금까지의 결정·근거·상태를 정리한 문서.
> 산출물 파일은 같은 폴더(`brands/`, `examples/`)와 PR #5에 있음.

## 1. 무엇을 하고 있나
- 신제품 **퓨어 베이비 츄잉**(4개월+ 퍼피용, 미니 5~6cm, 35g/200g, 첫 우피껌) 상세페이지를,
  기존 성견용 상세페이지 톤앤매너를 유지하면서 신제품에 맞게 재설계.
- 시작 계기: KEZ ProductPage 스킬(`HarinJin/KEZ-ProductPage-skill`) 검토 → 그 방식(패턴+브랜드값
  2층)을 펫포커스에 접목하는 흐름으로 확장.

## 2. 확정된 방향 (중요)
- **톤앤매너 = "절충"**: 기존의 컬러풀 다색 섹션·친근한 말투·이모지·말풍선·`Point/Check Point`
  라벨·알약 뱃지·풍부한 실사를 **계승**하고, 배경만 **파스텔**로 낮춘다.
  (한때 "미니멀 단색 프리미엄"으로 갔다가 "기존과 너무 다르다" 피드백 받고 되돌림 — 반복 주의.)
- **대표 선호 무드**: "천연 단일원료 프리미엄"(네츄럴EX 벤치마크). 단, 색을 갈아엎는 게 아니라
  원물·직공장 서사를 상단으로 끌어올려 충족.
- **[핵심 서사] 순수 우선(Pure-first)**: 주력은 **퓨어플레인(순수 우피껌)**. 닭안심·황태·첨가는
  기호성 때문에 팔 뿐. "첫 입맛이 평생 간다 — 첨가부터 먹은 아이는 순수 껌을 거부하기도, 아기 때부터
  순수만 먹은 아이는 향 없이도 잘 먹는다." → 베이비=순수 입맛의 출발점, 베이비와 퓨어플레인을 잇는 다리.
  (효능 단정 아님, 기호성 길들여짐이라는 경험담으로 서술.)

## 3. 근거가 된 자료
- **실제 패키지·제품 촬영본**: 크라프트 봉투 + 세이지그린 라벨 → 베이비 라인 액센트(세이지/크라프트).
  실제 컷(스틱·패키지)은 목업에 삽입됨.
- **쿠팡 마케팅 분석(2026-07-22)**:
  - 7월 매출 916만, 마케팅비 37.5%(광고 30.1%=276만, 쿠폰 7.4%), 순이익률 29.6%.
  - **CTR 0.19%(병목)** / CVR 12.01%(상품력 좋음) / 매출 71.7% 광고 의존.
  - 처방: 대표 썸네일 실물컷 70%+[100% 한우 생원물][30년 장인 직공장] 뱃지, 상세 상단에 직공장 스토리,
    광고비 30%→15~20%. → 상세엔 "차별화 서사를 상단으로"로 반영(직공장 공정·인증은 기존에 이미 있음).
- **경쟁 벤치마크(네츄럴EX 터키츄)**: "천연 단일원료·오래 씹는" 포지션 = 우리와 정면.
  우위 = 수입 칠면조 힘줄 대신 **국내산 한우 생우피**, 외주 대신 **자가 직공장**.

## 5. Figma 실그리기 상태 (미완)
- 이 세션(웹)엔 **Figma MCP 미연결**. 웹 세션은 로컬 Figma(localhost)에 접근 불가.
- 실그리기하려면 **Figma 있는 PC에서 로컬 Claude Code CLI 실행**이 정석.
- **블로커**: KEZ 스킬은 `use_figma`/`figma-use`라는 **쓰기용 브릿지**에 의존하는데, 그 조각이
  KEZ 저장소에 공개돼 있지 않음 → 제작자(HarinJin)에게 설치법 확인 필요. 그 전엔 Figma 생성 불가.
