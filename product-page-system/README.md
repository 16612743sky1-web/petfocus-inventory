# 상세페이지 디자인 시스템 (product-page-system)

펫포커스 상세페이지를 **KEZ ProductPage 방식(패턴 + 브랜드 값 2층 구조)**으로
일관되게 생성·재생성하기 위한 브랜드 자산 모음이다. 이 폴더는 이 저장소의 다른 도구
(현장재고 `index.html`, `stock-cost/`, `detail/`)와 독립적이며, 상세페이지 제작의
**"값(values)" 층**만 담는다.

## 구조

| 경로 | 내용 |
|---|---|
| `brands/petfocus.md` | 펫포커스 브랜드 프로파일 — 색(역할→HEX)·폰트·섹션 순서·카피 톤·이미지 규칙·제품 팩트 |

> **패턴 층(design-dna)**은 여기 두지 않는다. 색/타이포/레이아웃/카피의 "패턴"은
> KEZ 스킬(`HarinJin/KEZ-ProductPage-skill`) 또는 향후 이 폴더에 추가할 패턴 문서가 담당하고,
> 이 폴더의 브랜드 프로파일이 거기에 **실제 값**을 채운다.

## 어떻게 쓰나

1. **HTML 목업 / 상세페이지 카피 작업 시** — `brands/petfocus.md`의 색·폰트·섹션·카피 규칙을
   기준으로 삼는다. (마케팅 톤앤매너는 `.claude/skills/marketing`와 함께 본다.)
2. **Figma에 직접 프레임 생성(KEZ 스킬)** — 이 브랜드 프로파일을 스킬에 물려 오토레이아웃
   프레임을 생성한다.
   ⚠️ **전제**: Figma가 실행 중인 PC에서 **Claude Code를 로컬로 실행**해야 한다.
   클라우드/웹 세션은 로컬 Figma MCP(localhost)에 접근할 수 없다. (또는 claude.ai 공식
   Figma 커넥터 연결 — 단 공식 MCP는 주로 읽기 위주라 "프레임 쓰기" 가능 여부는 확인 필요.)

- 쿠팡 마케팅 분석(2026-07-22) → CTR 0.19% 병목, "30년 직공장·100% 한우 생원물·유통거품 ZERO"
  차별화를 상단에 배치.

  국내산 한우 생우피 우위 강조.
