# 브랜드 프로파일 — 펫포커스 (PetFocus)

> KEZ ProductPage 스킬의 **"값(values)" 층**. 패턴은 `design-dna/*`가 담당하고,
> 이 문서는 펫포커스의 확정 HEX·폰트·치수·문구 규칙을 **역할→값**으로 기록한다.
> 값이 미정인 슬롯은 `[미정]`으로 두고 제작 전 사용자에게 확인한다.

> **기준 원칙(2026-07-24 정정)**: 신제품(베이비 라인)은 **기존 성견용 상세페이지의 톤앤매너를
> 유지하면서 파스텔로 낮추는 "절충"**이다. 미니멀·단색 프리미엄으로 리브랜드하는 것이 **아니다.**
> 기존의 컬러풀 섹션·친근한 말투·이모지·말풍선·`Point/Check Point` 라벨 시스템·알약 뱃지·풍부한
> 실사를 **그대로 계승**하고, 배경색만 연한 파스텔로 바꾼다. 대표 선호(단일원료 프리미엄)는
> **색을 갈아엎어서가 아니라, 원물·직공장 서사를 상단으로 끌어올려** 충족한다.

---

## 0. 인터뷰 확정값

- **브랜드**: 펫포커스 — 프리미엄 한우 우피껌 자체 생산·판매(2인, 형/동생).
- **핵심 무기**: ① 100% **국내산 한우 생우피** 단일원료 ② **부모님 30년 가공 직공장 Direct**
  (유명 펫 브랜드 원물 공급 이력) ③ **유통거품 ZERO**. → 기존 페이지 Point 03(청결 제조)·
  Point 04(인증)로 이미 증빙 존재. 쿠팡 분석 과제는 "이 서사를 **상단으로 이동**"이다.
- **경쟁 포지션**: 네츄럴EX 터키츄 등 "천연 단일원료·오래 씹는" 개껌군과 정면 경쟁.
  우위: 수입 칠면조 힘줄이 아니라 **국내산 한우 생우피**, 외주가 아니라 **자가 직공장**.
- **판매 채널**: 쿠팡 브랜드스토어, 네이버 스마트스토어. 모바일 우선(폭 ~520px 세로 스크롤).
- **규제선**: 효능 **단정 금지**("치석 관리에 도움"까지). 무첨가·국내산은 사실이므로 명시.
- **판매 구성**: 전 제품 **단품**(세트 없음). 특정 용량 단정 문구 지양.

## 1. 색 프로파일 (01-color 대응)

### 기존 브랜드 색 세계(계승 대상)
성견용 상세페이지는 **섹션마다 배경색이 바뀌는 컬러풀 구성**이다: 하늘색(커버)·비비드 블루
(option)·핫핑크/보라/핑크(실사 플랫레이)·회색(공감)·옐로우(Check Point)·크림/베이지(무첨가).
포인트 텍스트는 **파랑·빨강** 강조. 이 "다색 섹션 + 굵은 흰 글씨" 구조가 브랜드 인상이다.

### 베이비 라인 = 위 색들을 **파스텔로 낮춘 버전** (톤: 따뜻·부드럽게)
| 기존(성견) | 베이비 파스텔 | HEX(안) |
|---|---|---|
| 하늘색 커버 | 연하늘·크림 | `#DCEBF5` / `#FAF7F0` |
| 비비드 블루 option | 연블루 | `#CFE0EF` |
| 핫핑크 플랫레이 | 베이비핑크·피치 | `#FBE1E1` / `#FBE7DB` |
| 보라 플랫레이 | 라벤더 | `#E8E2F6` |
| 옐로우 Check Point | 버터 | `#FDF3D3` |
| 크림/베이지 무첨가 | 크림 | `#FAF7F0` |
| 패키지 세이지/크라프트 | 베이비 라인 액센트로 | `#3F6B4F` / `#B98F5E` |

- **① 중립축(저채도 웜)**: 본문 `#3A352D`, 캡션 `#8B8171`, 역상 `#FAF6EE`.
- **② 액센트**: 강조 텍스트는 기존처럼 **파랑 계열**(`#3C6FB0`) + 베이비 패키지 **세이지**(`#3F6B4F`)
  를 함께. 빨강(`#C0492F`)은 "믿을만한 간식 찾기 어려우셨죠?" 같은 **공감·금지** 강조에.
- **③ 의미색**: 긍정=세이지/파랑, 경고·금지아이콘=붉은 계열, 특수강조=버터옐로.
- **뱃지**: 기존 **알약형 + 이모지** 유지, 색만 파스텔로(저지방·최고급 한우·첨가물 0%·100% 수작업).

> 핵심: "예쁜 단색"이 아니라 **기존의 다색 리듬을 파스텔로 유지**. 섹션 전환의 색 변화가 브랜드다움.

## 2. 타이포그래피 (03-typography 대응)

- **폰트**: Pretendard/산돌 계열 — **Bold~Black(제목)** / Regular(본문). 영문 악센트 소문자 트래킹.
- 컬러 배경 위 **굵은 흰 글씨** 큰 타이틀(기존 커버·option 방식) 유지.
- 위계: 커버 타이틀 40~44 / 섹션 제목 28~34 / 리드 15~17 / 본문 14~15 / 캡션 12~13.
- 숫자 `tabular-nums`. 파스텔이라도 **여백 과하게 비우지 말 것**(기존은 정보 밀도 있음).

---

## 3. 섹션 구성·순서 (05-page-purpose 대응)

기존 순서를 계승하되, 쿠팡 분석대로 **원물·직공장 서사를 상단으로** 당긴다.

1. **커버** — 큰 타이틀 + 강아지 실사 + **알약 뱃지 4개**. (베이비: "우리 아이가 처음 만나는 천연 한우 우피껌", 4개월부터 OK)
2. **option** — 용량/형태 비교(단품, 동그란 라벨 유지)
3. **공식 판매처** 고지 배너
4. **원물·전문성 (기존 Point 01 "전문가가 만드는 최고급 한우 우피껌"을 상단으로)** — 유리컵 원물 컷 + "단백질·콜라겐 풍부"
5. **"믿을만한 간식 찾기 어려우셨죠?" → 직공장 도입** (기존 흐름 계승, 위로)
6. **Point "청결한 시설에서 안심 제조"** — 기존 **공정 실사**(세척→건조→자르기→밀봉) 그대로
7. **무첨가** — 빨간 금지 아이콘 6종(향미증진제·착색료·감미료·글루텐·향료·보존료)
8. **공감 말풍선** — 보호자 상황(베이비: 이앓이·첫 츄잉 등). 말풍선·이모지 유지
9. **급여 장면 그리드** + PET FOCUS/CHEWING/ENJOY 스티커 도장
10. **인증** — 사업자·사료제조업·도축검사·사료성분 등록증 스캔
11. **Check Point "크기·모양 달라도 중량 같아요"** — 스틱 실사 + 노란 말풍선 안내
12. **Check Point "일일 권장 급여량 및 안내사항"** — 체중별 표 + 각주

## 4. 카피 톤앤매너 (04-copy 대응)

- 과장 없는 **다정한 존댓말**("~있어요/~해요"). 친근·귀여움 유지(이모지 OK, 남발만 주의).
- **문장 길이 섞기**, 빈 강조어 삭제, 번역투 제거. 추상→구체(수치·식감).
- 효능 **단정 금지**. 원료·원산지·제조는 사실이므로 단정 OK.
- **말풍선·"Point 01~04"·"Check Point"** 라벨 어투를 기존 그대로 계승.

### 확정 헤드카피 자산
- 커버: **"우리 아이가 처음 만나는 천연 한우 우피껌"** / `4개월부터 OK`
- 원물: **"전문가가 만드는 최고급 한우 우피껌"**(기존) — 베이비엔 "처음"을 얹어 변주
- 직공장: **"유명 브랜드가 선택한 그 원물, 부모님 공장에서 바로."**
- 무첨가: **"몸에 해로운 원료는 사용하지 않아요"**(기존)
- 대조: **"수입 힘줄 대신 국내산 한우 생우피."**

## 5. 이미지 처리 (06-image-treatment 대응)

- **실사 촬영본 우선.** 기존 페이지에 **공정 컷·급여 장면·인증 서류·플랫레이가 이미 다수 존재** →
  베이비 라인도 최대한 재활용/재촬영. AI 이미지에 **한글 굽지 않기**(편집 툴 별도 레이어).
- 손그림 장식·**스티커 도장(PET FOCUS/CHEWING/ENJOY)**·강아지 실사 합성 등 **기존 장식 언어 계승**.
- 뱃지: 알약형 + 이모지 유지(파스텔 색).

## 6. 제품 팩트

- 공통: 100% 국내산 한우 우피, **단일원료**, 무첨가·무방부제·색소/향미제 무첨가, 국내 생산.
- 제품군: 퓨어플레인츄잉(35/70g), 닭안심 회오리츄잉(35/70g), 건조닭안심(30/50g).
- **퓨어 베이비 츄잉**: 4개월 이상 퍼피용, **미니 5~6cm**, 35g/200g. 첫 우피껌 포지셔닝.
- 성견 스틱 규격: 길이 12~15cm·둘레 2~3cm(모양·크기 달라도 중량 동일).
- 급여량 표(기존): 3~6개월/1~5kg/6~10kg/11~25kg/25kg↑ → 10~20g/20~30g/30~40g/40~70g/70g↑.
  ※ 개체 특성·건강·운동량·계절에 따라 다를 수 있음(단정 아님).
