## 1. 영업장·구역 관리 (교차오염 방지) ★가장 중요
수제 소량 생산에서 심사가 가장 많이 걸리는 부분. **더러운→깨끗한 한 방향 동선**이 핵심.

- [ ] ★ 작업장이 **일반구역(원료보관·전처리·세척)**과 **청결구역(건조·내포장)**으로 구분되어 있다 (벽·칸막이·공간 분리 또는 시간차 구분+기록)
- [ ] ★ 원료 입고 → 전처리 → 건조 → 포장 → 완제품 보관이 **한 방향으로** 흐르고, 역류·교차가 없다
- [ ] 작업장 바닥·벽·천장이 **세척·소독이 쉬운 재질**(방수, 균열 없음)이다
- [ ] 작업장과 화장실·생활공간이 분리되어 있다
- [ ] 출입구에 **손 세척·소독 설비**(또는 발판소독조)가 있다
- [ ] 채광·조명이 충분하고, 조명에 **파손 대비 보호커버**가 있다
- [ ] 환기시설이 있어 습기·냄새·분진이 배출된다 (건조 공정 특성상 중요)

## 2. 방충·방서 (해충·설치류 차단) ★
한우 우피 = 유인성 강한 원료라 심사관이 반드시 본다.

- [ ] ★ 출입문·창문·배수구에 **방충망·에어커튼·트랩** 등 차단설비가 있다
- [ ] 방충·방서 **점검 기록**을 주기적으로 남긴다 (포충등 확인, 쥐덫 위치도 등)
- [ ] 외부와 통하는 개구부에 틈이 없다

## 3. 개인위생 관리
- [ ] 작업자(형·동생) 전용 **위생복·위생모·장화**가 있고 착용한다
- [ ] 작업 전/오염작업 후 **손 세척·소독** 절차가 정해져 있다
- [ ] 건강상태 이상(설사·화농성 상처 등) 시 작업 배제 규칙이 있다
- [ ] 작업장 내 흡연·취식·장신구 금지 규칙이 게시되어 있다

## 4. 시설·설비 위생
- [ ] 작업대·칼·도구가 **세척·소독 가능한 재질**(스테인리스 등)이다
- [ ] **건조기**를 정기 청소·점검하고 기록한다 (온도 관리 가능 여부 확인)
- [ ] 원료·완제품이 **바닥에 직접 닿지 않게** 팔레트·선반에 보관한다
- [ ] 세척·소독제를 식품/사료용 구역과 분리 보관하고 라벨을 붙인다
- [ ] 폐기물·부산물 전용 용기가 있고 뚜껑이 있다

## 5. 원료·부자재 관리 (입고~보관)
- [ ] ★ **한우 우피 원료의 입고 검수 기준**이 있다 (공급처, 이물·부패·이취 확인, 성적서/거래명세 보관)
- [ ] 식초·포장재 등 부자재도 **입고 확인·보관 기준**이 있다
- [ ] 원료별 **선입선출(FIFO)** 및 유통기한/입고일 표시로 관리한다
- [ ] 냉장·냉동이 필요한 원료는 **온도관리·기록**을 한다
- [ ] 부적합 원료를 **격리·표시**하는 자리가 있다

## 6. 용수 관리
- [ ] 세척·제조에 쓰는 물이 **먹는물 수질기준**에 적합하다 (상수도면 문제없음, 지하수면 수질검사 성적서 필요)
- [ ] (지하수 사용 시) **정기 수질검사** 기록이 있다

## 7. 완제품 보관·표시·운송
- [ ] 완제품 보관 구역이 청결하고 온·습도가 관리된다 (건조제품 눅눅해지지 않게)
- [ ] 제품 **표시사항**(사료 표시기준: 제품명·원료·중량·제조원·제조일 등)이 규정에 맞다
- [ ] 출고·배송 중 오염·파손 방지 포장이 되어 있다

## 8. 검사·모니터링
- [ ] 완제품 **수분·미생물 등 자가 또는 위탁 검사**를 하거나 계획이 있다 (건조 간식의 핵심 품질지표)
- [ ] 검사 결과 **기록·보관** 체계가 있다

## 9. 회수(리콜) 관리
- [ ] 문제 발생 시 **회수 절차·연락체계**가 문서로 있다
- [ ] 제품 **생산·판매 이력 추적**이 가능하다 *(참고: 펫포커스 재고앱의 입출고·이력 데이터가 이 근거로 활용 가능)*

---

## 10. HACCP 본체 (위해요소 분석 + CCP)
선행요건이 갖춰진 뒤, 우피껌 공정의 위해요소를 분석하고 중요관리점(CCP)을 정한다.

- [ ] 공정 흐름도(원료입고→세척→절단→건조→금속검출→포장→보관)를 작성했다
- [ ] 각 공정의 **위해요소**(생물학적=미생물/화학적=잔류물/물리적=이물·금속)를 분석했다
- [ ] ★ **CCP 후보를 정했다** — 우피껌 전형적 후보:
  - **건조 공정**: 최종 수분·가열로 미생물 제어 (온도·시간·수분 한계기준 설정)
  - **금속검출**(도입 시): 물리적 이물 관리
- [ ] 각 CCP의 **한계기준·모니터링 방법·개선조치·검증·기록**을 정했다 (HACCP 7원칙)

## 11. 문서·기록 (심사 제출용)
- [ ] HACCP 관리계획서(선행요건 프로그램 포함)를 작성했다
- [ ] 아래 **기록 서식**을 갖추고 실제로 기록 중이다:
  - [ ] 개인위생 점검일지
  - [ ] 작업장 청소·소독 일지
  - [ ] 방충·방서 점검일지
  - [ ] 원료 입고검수 일지
  - [ ] 건조(CCP) 모니터링 일지 (온도·시간·수분)
  - [ ] 완제품 검사·출고 기록
- [ ] HACCP 팀(형·동생 역할 분담)과 **교육 이수** 계획이 있다
